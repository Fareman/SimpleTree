# SimpleTree

Простая реализация заполнения и считывания бинарного дерева.

## Исходные данные

Целочисленный массив с уникальными значениями.

## Требования для запуска

1. Среда исполнения .Net 8
2. Собрать решение в корневой папке проекта командой `dotnet build`
3. Запустить приложение в корневой папке проекта командой `dotnet run`
